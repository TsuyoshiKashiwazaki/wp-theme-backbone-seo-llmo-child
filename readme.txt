Backbone SEO LLMOテーマの子テーマサンプル。親テーマを安全にカスタマイズできます。

== Description ==

Backbone SEO LLMO Childは、Backbone SEO LLMOテーマの公式サンプル子テーマです。

= 子テーマとは？ =

子テーマは親テーマの機能を継承しながら、カスタマイズを安全に行うための仕組みです。

= メリット =

* 安全な更新: 親テーマを更新してもカスタマイズが消えない
* 簡単なカスタマイズ: 必要なファイルだけを編集すればOK
* 元に戻せる: 子テーマを無効化すれば元の状態に戻る
* 学習に最適: 親テーマのコードを参考にしながら学べる

= 使い方 =

1. 外観 > テーマ から「Backbone SEO LLMO Child」を有効化
2. 外観 > テーママニュアル からカスタマイズ方法を確認
3. style.css にカスタムCSSを追加
4. functions.php にカスタム関数を追加
5. 必要に応じてテンプレートファイルをカスタマイズ

詳しい使い方は、管理画面の「外観 > テーママニュアル」をご覧ください。

== Installation ==

1. 親テーマ「Backbone SEO LLMO」をインストール
2. 子テーマをインストール（既にインストール済みの場合はスキップ）
3. 外観 > テーマ から「Backbone SEO LLMO Child」を有効化

== Frequently Asked Questions ==

= 子テーマに全ファイルをコピーする必要がありますか？ =

いいえ！必要ありません。カスタマイズしたいファイルだけをコピーしてください。

= 親テーマを更新したら子テーマの設定は消えますか？ =

いいえ、消えません。それが子テーマの最大のメリットです。

= 子テーマのファイルが優先されますか？ =

はい。同じ名前のファイルがある場合、子テーマのファイルが優先されます。

= functions.php は上書きされますか？ =

いいえ。functions.php だけは特別で、親と子の両方が実行されます。

= カスタマイザーの設定はどうなりますか？ =

カスタマイザーの設定は親テーマと子テーマで共有されます。
